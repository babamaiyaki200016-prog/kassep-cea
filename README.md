# KASSEP cost-effectiveness analysis

Analysis code for: *Cost-effectiveness of a community-based sample registration
system with verbal and social autopsy for maternal mortality surveillance in
Kano State, Nigeria.*

Written in **R**. Every number and every figure in the paper is generated by this
code. Nothing is hand-entered.

## Headline result

**US$1,824 (₦2,516,602) per maternal death detected and cause-assigned**, from the
perspective of the Kano State Government. This is also the incremental
cost-effectiveness ratio, because the status-quo CRVS produces no cause-assigned
maternal deaths at no attributable marginal cost. This exceeds the
willingness-to-pay threshold of one times GDP per capita (US$1,223): the
probabilistic sensitivity analysis puts the probability KASSEP is cost-effective
at this threshold, State perspective, at 2.5%. The threshold would need to reach
approximately US$1,850 for the base case to be cost-effective with 50%
probability.

## Reproduce

```r
Rscript install_deps.R    # ggplot2, scales, jsonlite
Rscript run_all.R         # model + DSA + PSA + all four figures
```

Or step by step:

```r
Rscript R/01_model.R      # base case, one-way DSA, 10,000-draw PSA
Rscript R/02_figures.R    # Figures 1, 2, A1, A2
```

The seed is fixed (`set.seed(20260712)`), so the probabilistic results are exactly
reproducible. Base-case and one-way results are analytic and carry no simulation
error.

## Method in brief

Single-cycle decision tree; the unit of analysis is one live birth in the
surveillance population. Perspectives: Kano State Government (primary) and full
economic (secondary). Discount rate 5%. Costs in 2026 naira and US dollars at
₦1,380/US$. Willingness to pay: 1 × GDP per capita = US$1,223. All epidemiological
inputs are **unweighted**. Reported per CHEERS 2022.

Probabilistic analysis: 10,000 Monte Carlo draws. Costs ~ Gamma (CV 0.20);
maternal deaths ~ Poisson(93); cause-assignment probability under KASSEP ~
Beta(94, 1) (the posterior for 93/93 successes under a uniform prior); under the
status quo ~ Beta(1, 199); exchange rate ~ lognormal (SD 10%).

## Data availability

Individual-level surveillance records are **not** in this repository: they contain
identifiable information about deceased women and their households. The aggregate
parameters in `data/parameters.json` reproduce every result in the paper.
